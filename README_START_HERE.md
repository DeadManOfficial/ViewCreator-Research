## üí° TIPS FOR SUCCESS

### For First-Time Users
1. **Start small**: Produce one segment completely before attempting full season
2. **Don't modify prompts**: They're optimized through testing, use as-is
3. **Record extra takes**: Easier to have options in editing than re-record
4. **Follow the schedule**: 5-day production cycle prevents burnout
5. **Archive everything**: You'll want to reference/reuse assets later

### Common Mistakes to Avoid
- ‚ùå Modifying Sora prompts "to make them better" (they're already optimized)
- ‚ùå Over-processing voice (the distortion is deliberate, not excessive)
- ‚ùå Skipping film grain overlay (essential to aesthetic)
- ‚ùå Using smooth transitions (breaks the harsh broadcast style)
- ‚ùå Shouting the dialogue (DEADMAN is controlled, not emotional)

### Quality Over Speed
- Don't rush to complete all 8 segments at once
- One excellent segment per week is better than eight mediocre ones
- DEADMAN's credibility depends on production quality
- Your audience will tolerate weekly uploads better than rushed content

## üÜò SUPPORT & TROUBLESHOOTING

### If You're Stuck
1. **Check the workflow guide troubleshooting section** - covers common issues
2. **Review the specific phase** you're working on in the workflow
3. **Verify you're using the correct source document** for your task
4. **Check file naming conventions** - incorrect names cause confusion

### If Something Doesn't Work
- **Sora generation off**: Use exact prompt from library, don't modify
- **Voice sounds wrong**: Review processing chain in workflow, compare to reference
- **Video won't export**: Check codec settings match platform requirements
- **Timing is off**: Refer to master production timecode tables

### Quality Issues
- **Video looks flat**: Apply color grading and film grain per loop notes
- **Voice unintelligible**: Reduce distortion, increase 2-4kHz presence
- **Doesn't feel like DEADMAN**: Review character guidelines in voice scripts
- **Pacing feels wrong**: Record at 140-160 words per minute
