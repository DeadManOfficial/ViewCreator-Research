# SORA 2 INTERACTIVE PROMPT BUILDER

**Version:** 2.0  
**Purpose:** Step-by-step prompt construction tool  
**Time to Complete:** 5-10 minutes per prompt

---

## HOW TO USE THIS BUILDER

1. Work through each section sequentially
2. Fill in only what's relevant to your shot
3. Copy the final assembled prompt
4. Iterate based on results
