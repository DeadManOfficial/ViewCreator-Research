## Multi-Agent Workflows

**Content Pipeline:**
```
/ideas [topic] -> /seo [best headline] -> /email [promote article]
```

**Video Pipeline:**
```
/clip [video] -> /visual [thumbnail] -> /sora [trailer]
```

**Creative Pipeline:**
```
/story [concept] -> /mj [scene visuals] -> /sora [video scenes]
```

## Usage
Type any command with your topic/input to spawn that agent.
