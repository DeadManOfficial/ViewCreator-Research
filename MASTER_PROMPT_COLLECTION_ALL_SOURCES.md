#### Block Definitions

**Block 1: HOOK_SCENE_DESCRIPTION (0-2 seconds)**
- Visually arresting, high-impact opening
- Focus: Extreme close-up, dynamic motion, or immediate visual paradox
- Goal: Maximum audience retention

**Block 2: BEAT_1_ACTION (Setup)**
- Primary action or character introduction
- Focus: Clear subject, defined movement, environmental interaction
- Goal: Establish narrative foundation

**Block 3: BEAT_2_ACTION (Conflict/Development)**
- Narrative development or conflict escalation
- Focus: Camera angle change, secondary element introduction, action escalation
- Goal: Drive toward payoff

**Block 4: PAYOFF_SCENE_DESCRIPTION (Climax/Resolution)**
- Final climactic scene, narrative arc resolution
- Focus: Final visual punch, thematic resolution, unexpected twist
- Goal: Memorable conclusion

**Block 5: STYLIZATION_BLOCK (Aesthetics)**
Required Elements:
- Core Aesthetic: `:cybercore`, `:vaporwave`, `:horrorcore`

- Lighting/Color: `Volumetric lighting`, `deep shadows`, `neon magenta and cyan palette`
- IP-Safe Parody: `No real people`, `no photoreal faces`, `no brands`, `highly stylized cartoon physics`

**Block 6: TECHNICAL_BLOCK (Camera & Production)**
Required Elements:
- Camera: `35mm lens`, `anamorphic`, `deep depth of field`, `low angle`
- Movement: `Slow smooth 360-degree dolly shot`, `sudden whip pan`, `cinematic crane shot`
- Resolution/Aspect Ratio: `4K`, `16:9 cinematic aspect ratio`
- Duration: `5.0 seconds`

### Quality Hierarchy

**Tier 1 (Highest Quality):**
- Full 8-component structure
- Technical metadata (ARRI Alexa 65, Zeiss lenses)
- EXIF-style realism boosters (_DSC09421.ARW)
- Specific lighting ratios (20:1 contrast)
- Beat-by-beat action timeline

**Tier 2 (High Quality):**
- 5-6 components present
- Camera specs without exact models
- General realism tokens (Photorealistic, 4K)
- Action described but not beat-mapped

**Tier 3 (Good Quality):**
- 3-4 components
- Basic camera info (wide angle, close-up)
- Style references (cinematic, viral-ready)
- Simple action description

**Tier 4 (Functional):**
- 1-2 components
- Subject and basic setting only
- Minimal technical detail
- Relies on model defaults

### Common Mistakes to Avoid

1. **Overloading with conflicting styles** - Don't mix "Pixar-style 3D" with "gritty found footage"
2. **Vague action descriptions** - "Person walks" vs. "Person takes 3 steps forward in 2 seconds"
3. **Missing temporal consistency** - Always include "Perfect Continuity" for multi-shot sequences
4. **Ignoring aspect ratio** - Vertical content needs 9:16, not 16:9
5. **Forgetting negative prompts** - Specify what to avoid (watermarks, text, logos)
6. **No lighting specification** - "Natural lighting" is not enough, specify source and quality
7. **Generic realism tokens** - "Realistic" < "Photorealistic" < "Hyper-detailed, subsurface scattering"
