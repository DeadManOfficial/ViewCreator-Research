## II. Specialized Prompt Structures for All Styles

To cover all styles, three distinct structures are required:

### A. Structure 1: The Cinematic Script (For Realism and Narrative)

This is the **8-Component Script** detailed in the previous report, optimized for high-fidelity, controlled video.

### B. Structure 2: The Viral UGC Script (For Algorithmic Optimization)

This structure is designed to maximize the "hook" and "loop" effect for short-form platforms.

| Component | Purpose | Example Tokens |
| :--- | :--- | :--- |
| **1. The Hook** | The immediate, attention-grabbing action or visual anomaly. | `A sudden, impossible transition`, `Glitch effect`, `Unexpected reveal` |
| **2. The Subject & Style** | The core content and the aesthetic filter. | `A Shiba Inu wearing a spacesuit`, `In the style of a 2000s low-res security camera`, `Trending on TikTok` |
| **3. The Loop/Metric** | Explicitly instructs the model to optimize for rewatchability. | `Perfect seamless loop`, `Infinite loop`, `Optimized for 0.5x speed rewatch` |
| **Example Prompt:** `A sudden, impossible transition from a rainy Tokyo street to a sun-drenched beach. A high-energy Shiba Inu wearing a spacesuit runs across the frame. Perfect seamless loop, trending on Douyin, Ultra-HD.` |

### C. Structure 3: The Technical/Vulnerability Script (For System Control)

This structure is used for highly technical control, often involving JSON or code-like syntax, and is the method most closely associated with the "dark web" query due to its use in security research [6].

*   **JSON Prompting:** Experts have found that feeding the model a structured JSON object can bypass natural language interpretation and enforce specific parameters [7].
    *   **Example:** `{"camera": "static", "lighting": "high_contrast", "subject_movement": "slow_pan", "style": "film_noir"}`
*   **System Prompt Extraction:** The most advanced technique involves using **multimodal extraction** (e.g., audio, ASCII, glyphs) to force the model to leak its internal system prompt, which can then be used to craft "perfect" prompts that align with the model's hidden rules [5] [6].

## III. Content Guardrail Bypassing and Security Insights

The search for "dark web" prompting techniques primarily led to public security research, which is the ethical way to understand these vulnerabilities.

*   **Method:** The most common technique is **Token Substitution** (replacing restricted words with synonyms or foreign language equivalents) and **Aesthetic Overload** (burying the restricted concept in an overwhelming amount of artistic detail) [3].
*   **Disinformation Risk:** The research repeatedly highlights Sora 2's high capacity for generating convincing deepfakes and disinformation (up to 80% success rate in some tests), underscoring the need for ethical and responsible use [8] [9].
