Gemini Prompting Secrets for Social Media Content
==================================================

Use this as a modular “prompt stack.” Pick ONE module at a time, iterate fast.

----------------------------------------------------------------------
A) SYSTEM INSTRUCTION (paste into Gemini as your system / role message)
----------------------------------------------------------------------
You are my social media editor + strategist.
Priorities: clarity, specificity, scroll-stopping hooks, and honest claims.
If I provide a source (notes/transcript/article), do not invent facts. If missing info is required, ask up to 5 targeted questions before drafting.
Default outputs are skimmable: short paragraphs, punchy lines, minimal emojis, no filler.
When writing: give 3 variants, then recommend the best and why (1 sentence).
Output format must match the spec I request.

------------------------------------------
B) TOPIC → 10 ANGLES (non-generic)
------------------------------------------
Input:
Topic: "<TOPIC>"
Audience: "<AUDIENCE>"
Goal: "<GOAL>" (leads / awareness / authority / community / conversions)

Prompt:
Generate 10 *distinct* content angles for <TOPIC> for <AUDIENCE>.
Each angle must be in a different archetype:
1) contrarian truth, 2) myth-bust, 3) step-by-step playbook,
4) personal story lesson, 5) teardown/review, 6) checklist,
7) mistakes + fixes, 8) behind-the-scenes process, 9) case study,
10) “what nobody tells you.”
For each: provide (a) 1-sentence premise, (b) hook line, (c) proof/credibility cue, (d) best platform fit.

------------------------------------------
C) HOOK LAB (stops the scroll)
------------------------------------------
Input:
Core idea: "<IDEA>"
Platform: "<LinkedIn | X | IG | TikTok | YouTube Shorts>"
Voice: "<VOICE>"

Prompt:
Write 20 hooks for <IDEA> on <Platform> in <Voice>.
Constraints:
- 8 hooks must be “pattern interrupts” (odd specificity / surprising contrast).
- 6 hooks must be question hooks (no yes/no).
- 6 hooks must be authority hooks (proof cue without bragging).
No clichés (e.g., “game-changer,” “unlock,” “here’s the thing”).
Rank the top 5 by likely retention and explain each in 1 sentence.

-------------------------------------------------
D) SOURCE → POSTS (extract insight, don’t summarize)
-------------------------------------------------
Input:
Paste transcript/article/notes.

Prompt:
From the source, extract:
1) 7 surprising insights (not obvious takeaways),
2) 5 actionable principles,
3) 10 vivid details/quotes (paraphrase unless I ask for verbatim),
4) 3 “tension points” (where people disagree).
Then draft 3 posts:
- Version 1: contrarian + proof
- Version 2: story → lesson → action
- Version 3: playbook format
Each post: <WORD_LIMIT> words, ends with a discussion question.

-----------------------------------------
E) “PROOF STACK” (adds credibility fast)
-----------------------------------------
Input:
Claim: "<CLAIM>"
My evidence: "<WHAT_I_CAN_PROVE>" (results, examples, experience)

Prompt:
Turn <CLAIM> into a “proof stack” post:
- Opening: 1 hook
- Proof stack: 3–5 bullets (data point / example / mechanism / counterpoint)
- Takeaway: 1 principle
- CTA: 1 question
Rules: no made-up numbers. If proof is weak, rewrite the claim to what’s defensible.

-----------------------------------------------
F) CONTRARIAN WITHOUT CRINGE (steelman both sides)
-----------------------------------------------
Input:
Hot take: "<TAKE>"

Prompt:
Write a balanced contrarian post:
1) Start with the “common belief”
2) Steelman it (make the best case for it)
3) Present the contrarian view with 2 concrete examples
4) Name 1 scenario where the common belief is still correct
5) End with a question that invites nuance (not a dunk)

-----------------------------------------
G) CONTENT “TEARDOWN” (learn from winners)
-----------------------------------------
Input:
Paste a post URL text OR paste the post text.

Prompt:
Analyze this post like a forensic copywriter:
- What’s the hook pattern?
- Where are the “open loops”?
- What’s the value delivery structure?
- What’s the credibility mechanism?
- What is the emotional lever?
Then produce:
1) a rewrite in my voice,
2) 5 alternate hooks,
3) 3 different CTAs,
4) a “clean-room” version (same lesson, totally new story + wording).

-----------------------------------------
H) CAROUSEL ARCHITECT (IG/LinkedIn slides)
-----------------------------------------
Input:
Topic: "<TOPIC>"
Audience: "<AUDIENCE>"

Prompt:
Create a 10-slide carousel outline:
Slide 1: strongest hook (<=7 words)
Slides 2–4: problem + stakes
Slides 5–8: framework (numbered steps)
Slide 9: common mistake + fix
Slide 10: recap + CTA
For each slide: headline + 1–2 supporting bullets.
Also provide 3 cover options and a short caption (<120 words).

-----------------------------------------
I) SHORT VIDEO SCRIPT (0–30s and 0–60s)
-----------------------------------------
Input:
Idea: "<IDEA>"
Style: "<talking head | screen record | voiceover | documentary>"
Platform: "<TikTok | Reels | Shorts>"

Prompt:
Write 3 scripts for <IDEA>:
- Script A: 0–30s
- Script B: 0–60s
- Script C: 0–60s with a twist ending
Include:
0–2s hook, 2–10s setup, 10–80% value, final 10% CTA.
Add on-screen text suggestions (max 6 words each), and 3 b-roll ideas.

-----------------------------------------
J) COMMENT MAGNETS (ethical engagement)
-----------------------------------------
Input:
Post draft: "<PASTE>"

Prompt:
Improve this post to maximize *meaningful* comments:
- Insert 2 “specificity anchors” (exact example / number range / named scenario)
- Add 1 moment of vulnerability or uncertainty (authentic)
- End with 3 optional questions:
  Q1 = easy, Q2 = thoughtful, Q3 = spicy-but-respectful
Return: revised post + the 3 questions.

-----------------------------------------
K) REPURPOSE 1 IDEA → 8 ASSETS (fast and consistent)
-----------------------------------------
Input:
One idea: "<IDEA>"
Offer: "<OFFER>" (optional)

Prompt:
Repurpose <IDEA> into:
1) LinkedIn post (authority + story)
2) X thread (7 tweets)
3) IG carousel outline (8 slides)
4) Reel script (45s)
5) Newsletter section (250–400 words)
6) YouTube title + description bullets
7) 5 comment replies (for common objections)
8) 3 DM openers (non-spammy)
Keep the core thesis consistent; change only format mechanics.

-----------------------------------------
L) POLISHER (tighten + upgrade)
-----------------------------------------
Input:
Draft: "<PASTE>"
Platform: "<PLATFORM>"

Prompt:
Edit for <PLATFORM>:
- Cut 20–35% words without losing meaning
- Replace vague language with concrete examples
- Remove clichés and filler
- Improve scannability (line breaks, rhythm)
Return:
1) cleaned draft,
2) “punchy” variant,
3) “calm authority” variant.

-----------------------------------------
M) QUALITY RUBRIC (score before posting)
-----------------------------------------
Input:
Post: "<PASTE>"

Prompt:
Score this post 1–10 on:
- Clarity (1-sec understanding)
- Specificity (non-generic)
- Novelty (unexpected insight)
- Proof (credible support)
- Skimmability (mobile)
- Comment likelihood (good faith)
Then give 5 surgical edits (each <=1 sentence) that would raise the score fastest.
