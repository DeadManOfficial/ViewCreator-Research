## Extraction Methodology

This collection was assembled using **Wide Research** (parallel processing) techniques:

1. **Source Discovery:** Identified all primary and secondary sources
2. **Automated Extraction:** Used browser automation to click through interactive galleries
3. **Parallel Processing:** Processed multiple data sources simultaneously
4. **Format Conversion:** Converted all prompts to multiple formats (MD, CSV, JSON, YAML)
5. **Comprehensive Documentation:** Created detailed documentation for each source
